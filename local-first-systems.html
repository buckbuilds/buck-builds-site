How it works

Keep the workflow inspectable.

  1. 01

    Use readable files

    Markdown, simple tables, folders, and JSON are easier to inspect than a black-box tool you have to trust blindly.

  2. 02

    Add small automation

    Scripts and dashboards should remove repeat work while keeping the important records easy to open.

  3. 03

    Bring in platforms only when they earn it

    Payments, hosting, public distribution, and backups can use external tools. The source of truth stays simple.
